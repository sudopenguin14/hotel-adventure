# Hotel Adventure 
⚠️ IMPORTANT ⚠️

Click on Releases and download the game for your Operating system.

On Windows: Download the file "hotel-adventure-win.exe" from releases. Double click to open it. You may get a security warning. This is fine as I am not a verified developer. Click on More info, then run anyway. Enjoy the game!

On GNU/Linux Download the "hotel-adventure-linux" file and open the terminal in the directory where you downloaded the game and type "./hotel-adventure-linux" to launch it

If you don't want to download the game, you can open the python file (hotel-adventure.py) and copy-paste the code in a code editor which works with python(eg. VS studio code, thonny etc.)
